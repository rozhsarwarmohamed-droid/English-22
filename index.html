🔍 Language Detective

👋 NEAR
Things close to you
this
these
Singular → this
Plural → these
👉 FAR
Things far from you
that
those
Singular → that
Plural → those
Near + Singular: "What is this?" (one kiwi, near)
Near + Plural: "What are these?" (many kumquats, near)
Far + Singular: "What is that?" (one drink, far)
Far + Plural: "I want one of those." (kebabs, far)
